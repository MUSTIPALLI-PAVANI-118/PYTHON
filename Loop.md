Looping Statements

Loops are used to repeat a block of code.

4.1️⃣ for Loop
Purpose

Used when number of iterations is known.

Syntax
for variable in sequence:
    statement

Example
for i in range(5):
    print(i)

4.2️⃣ while Loop
Purpose

Executes code as long as the condition is True.

Syntax
while condition:
    statement

Example
i = 1
while i <= 5:
    print(i)
    i += 1
